# eslintAutoFix

A sublime plugin.

## setting

### eslint_path

* Global installation path of eslint.

### show_panel

* Whether the output panel, default true.


e.g.

```
{
  "eslint_path": "/Users/matianming/nvm/versions/node/v4.2.4/bin/",
  "show_panel": true
}

```

## command

### `eslint_auto_fix`

## keymap

### `super+shift+,`
